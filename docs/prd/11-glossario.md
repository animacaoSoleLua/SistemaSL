## Termos de Negocio

| Termo | Definicao |
|-------|-----------|
| Animador | Responsavel por coordenar eventos e criar relatorios |
| Recreador | Auxilia o animador no evento |
| Administrador | Chefe ou gerente com acesso total |
| Relatorio de evento | Registro do que ocorreu no evento |
| Advertencia | Registro de ocorrencia negativa do membro |
| Curso | Treinamento interno para membros |

---

## Termos Tecnicos

| Termo | Definicao |
|-------|-----------|
| API | Interface de comunicacao entre sistemas |
| Endpoint | URL que recebe requisicoes da API |
| JWT | Token usado na autenticacao |
| CRUD | Operacoes de criar, ler, atualizar e excluir |
| MVP | Versao minima funcional |

## Conceitos do Sistema

| Conceito | Definicao |
|----------|-----------|
| Suspensao | Bloqueio temporario de trabalho por 1 mes |
| Feedback individual | Observacao sobre um membro dentro do relatorio |

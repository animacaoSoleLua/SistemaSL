## Descricao do Epic

Oferecer uma visao rapida dos relatorios para o administrador.

---

## US-025: Resumo por periodo

**Como** administrador
**Quero** ver um resumo por periodo
**Para que** eu tenha visao rapida

**Prioridade:** Should Have
**Estimativa:** M
**Epic:** Dashboard analitica
**Dependencias:** US-004

**Criterios de Aceitacao:**
1. Mostra total de eventos no periodo
2. Mostra total de relatorios enviados
3. Mostra media de avaliacao
4. Permite escolher periodo
5. Funciona no celular
